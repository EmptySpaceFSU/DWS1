Definitions

Chapter 2, Text

Headings
Examples: <h1>,<h6>
Used to denote a section headings, higher numbers mean subsections.
Paragraphs
Example: <p>
Used to denote a paragraph, breaking longer texts into smaller blocks.
Bold
Example: <b>
Makes text inside the tags bold.
Italic
Example: <i>
Used to make text inside the tags italic.
Superscript and Subscript
Example: <sup>
Makes the text inside either superscript or subscript.
White Space
Example: <p>
White space refers to the gap between lines.
Line Breaks
Example: <br>
Line breaks act as a sort of carriage return, dropping to the next line of text.
Horizontal Rules
Example: <hr>
Creates a horizontal line to breakup text.
Strong
Example:<strong>
Strong, very similar to bold, adds a text effect making the letters thicker.
Emphasis
Example: <emp>
Emphasis looks very similar to italics.
Quotations
Examples: <q>
Used to create quotes in text. "Like here" he said.
Abbreviations
Examples: <abbr>
Used to store the names of terms, and possibly search for them later.
Citations
Example: <cite>
Defines the title of work with the citation tag.
Definitions
Example: <dfn>
Defines a term.
Address
Example: <address>
Place an address into tags.
Insert and Delete
Example: <del> ,<ins>
Strikes a word out, and replaces it with another.
Strikethrough
Example: <s>
Strikes out the word in the tags.

Chapter 3, Lists

Ordered Lists
<ol>,<li>
Creates lists with numbering.
Unordered Lists
<ul>,<li>
Creates lists without numbering.
Definition Lists
<dl>, <dt>, <dd>
Used for definition lists, a specific role.
Nested Lists
A list found within another list.
